# 学习教程

技术入门、工具使用教程和最佳实践。

## 如何添加

在此目录下创建新的 `.md` 或 `.mdx` 文件。建议按技术领域使用子目录组织：

```
tutorials/
├── rust/          # Rust 相关教程
├── python/        # Python 相关教程
├── tools/         # 工具使用教程
└── ...
```

### 建议格式

```markdown
---
title: 教程标题
date: 2026-06-11
authors: [作者名]
---

## 目标
本教程的目标读者和预期收获。

## 前置知识
学习本教程需要的前置知识。

## 正文
教程的详细内容。
```
