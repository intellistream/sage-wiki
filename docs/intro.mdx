# Sage Wiki

欢迎来到 **Sage Wiki** —— IntelliStream 团队知识库。

这里存放团队的**技术成果、学习心得、教程指南**和**规范文档**，通过 Git 协作维护，人人皆可贡献。

## 内容导航

| 分类 | 说明 |
|------|------|
| [成果展示](/docs/achievements/) | 项目里程碑、技术产出、获奖记录 |
| [学习教程](/docs/tutorials/) | 技术入门、工具使用、最佳实践 |
| [技术心得](/docs/tech-notes/) | 踩坑记录、源码分析、架构设计 |
| [团队规范](/docs/standards/) | 代码规范、工作流程、Review 准则 |
| [资源汇总](/docs/resources/) | 推荐书单、工具清单、常用链接 |

## 如何贡献

1. Clone 本仓库
2. 在对应分类目录下创建 Markdown 文件
3. 提交 Pull Request 或直接 push 到 main 分支
4. 站点自动构建部署

:::tip 提示
左侧边栏可以快速导航到各个分类下的文章。
:::
